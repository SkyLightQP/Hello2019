# Hello2019
> 2019년까지 남은 시간을 카운트합니다.

[2019년.me](http://xn--2019-yy8p.me/)

## Build
```
$ git clone https://github.com/SkyLightQP/Hello2019.git
# 코드를 다운로드합니다.

$ npm run build
# 빌드를 진행합니다.
```

## Getting started
사전에 빌드가 필요합니다.
```
$ npm run start
# 서비스를 시작합니다.

$ npm run dev
# 개발 모드로 시작합니다.
```
